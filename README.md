# Junhyuk AI Portfolio

Public portfolio for manufacturing AI, industrial analytics, and full-stack dashboard projects.

This portfolio contains public demo repositories rebuilt from the architecture and workflows of real deployed manufacturing AI systems.
Production source code, production data, customer information, credentials, infrastructure details, logs, certificates, and private environment values are not included.

All public demo data is synthetic or anonymized.

## Project Summary

### ML Studio / Insight View

ML Studio / Insight View is a public demo of a manufacturing AI analytics platform.
It provides a local demo stack for analyzing manufacturing operation data, training AI models, monitoring anomaly detection results, checking supervised learning results, and reviewing threshold-based alerts.

Key implementation areas:

* React frontend dashboard
* Spring Boot backend API
* FastAPI AI server
* MongoDB demo schema and seed data
* Unsupervised anomaly detection flow
* Supervised learning result view
* Threshold alert monitoring
* AI operation status monitoring
* Public demo documentation and synthetic data policy

Repository: [ml-studio-insight-view-demo](https://github.com/junhyuk-99/ml-studio-insight-view-demo)

### CNC/MCT Analytics Dashboard Demo

CNC/MCT Analytics Dashboard Demo is a public demo of a manufacturing equipment analytics dashboard.
It focuses on equipment utilization, runtime/cuttime ratio, alarm history, machine status monitoring, daily trends, and KPI visualization.

Key implementation areas:

* React + TypeScript frontend
* Spring Boot backend API
* MongoDB demo data model
* Recharts-based dashboard visualization
* Equipment utilization analytics
* Runtime / CutTime cutting ratio calculation
* Alarm history analytics
* Public demo documentation and synthetic data policy

Repository: [cnc-mct-analytics-dashboard-demo](https://github.com/junhyuk-99/cnc-mct-analytics-dashboard-demo)

## Public Demo Policy

These repositories are not production source code dumps.
They are public demo projects rebuilt from real project architecture and workflows.

The following items are intentionally excluded:

* Production source code
* Production database connections
* Customer data
* Real equipment history
* Server IP addresses
* Private credentials
* Logs
* Certificates
* Private environment values
* Internal deployment scripts
