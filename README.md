## CNC/MCT Analytics Dashboard

A dashboard project for monitoring and analyzing CNC/MCT equipment operation data.

### Main Features

- Equipment utilization analysis
- Runtime and cutting-time analysis
- Alarm history analysis
- Equipment status monitoring
- Spindle operation status analysis
- Spindle cooling fan status analysis
- Tool usage analysis
- Role-based user and menu management

### Tech Stack

- Frontend: React
- Backend: Spring Boot
- Database: MongoDB
- Visualization: ECharts
